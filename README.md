这是一本逻辑书, 用于入门名为`逻辑语言`的一门编程语言

这门语言是来源于游戏 [Mindustry] 游戏中的一种类似汇编的语言,
用于控制游戏中的各种东西.

由于其可以和 '显示屏' '信息板' '按钮' '炮台' 等交互, 所以可以制作一些交互式工具

本书基于的游戏版本约为 146.0

欢迎对本书提出建议
